# Docs do Projeto

Documentação principal do runtime atual do DirectorAi.

## Stack vigente

- Frontend: Vite + React + React Router + Tailwind 4 + PWA
- Backend: Express 5 em TypeScript
- Auth: Better Auth
- Banco: PostgreSQL
- Storage: Vercel Blob
- Estado: SWR para dados remotos, Zustand para estado client-only
